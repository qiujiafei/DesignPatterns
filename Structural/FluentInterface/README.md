# 流接口模式 (Fluent Interface)
### 目的
用来编写易于阅读的代码，就像自然语言一样（如英语）

### 示例
>+ Doctrine2 的 QueryBuilder，就像下面例子中类似
>+ PHPUnit 使用连贯接口来创建 mock 对象
>+ Yii 框架：CDbCommand 与 CActiveRecord 也使用此模式
