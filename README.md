## 环境和依赖
```$xslt
php 7.3
composer 1.7.3
phpunit 8.2.4
```
## demo使用
在根目录运行 `composer install`  
用例在每个设计模式对应的 `Test` 文件夹   
```shell
vendor/bin/phpunit Behavioral/Visitor/Tests/VisitorTest.php
```
